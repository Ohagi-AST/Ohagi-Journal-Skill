> 母版规则 7 文体层的展开，**仅在日语轨（投稿/修論）启用**。
> 解决用户核心痛点：以中文构思、转写成日文后残留的三类「中文味」。逐关过：①字形 → ②词汇/术语 → ③表达/语序/标点。

## 三关总览

| 关 | 残留类型 | 例 | 工具 |
|---|---|---|---|
| **①字形** | 中文简/繁字形冒充日文 | 經濟發展・经济发展 → 経済発展 | `zh_glyph_check.py`（确定性脚本） |
| **②词汇** | 中文词/中式日语（含**同形异义陷阱**） | 货币政策 → 金融政策；显著 → 有意 | 本文对照表 + 人工 |
| **③表达** | 中文直译腔（语序・范畴词・连接・标点） | 起到重要作用 → 重要な役割を果たす | 本文规则 + 人工 |

字形是机械的、可脚本化；词汇和表达需理解语义，**脚本只兜字形，词汇/表达必须人工逐句过**。

## 第①关 字形（脚本兜底）

中文简体/繁体字形看着像日文、实为中文字（經濟/经济 ≠ 経済）。用确定性脚本揪：

```bash
python skills/journal-drafting/scripts/zh_glyph_check.py draft.txt          # 报告残留字+行列位置
python skills/journal-drafting/scripts/zh_glyph_check.py draft.txt --fix     # 另存 draft.fixed.txt
echo "經濟發展と環境問題" | python skills/journal-drafting/scripts/zh_glyph_check.py -   # stdin
```

- 表驱动（`ZH2JP`：简体+繁体异形 → 日本新字体），自动剔同形字防误报，可按需扩 `ZH2JP`。
- ⚠️ **`--fix` 后必须人工复核引用区/专名区**：中文文献题名、中国人名地名（深圳・北京・《经济研究》）应**保留中文原字**，脚本不区分会一并替换。
- 脚本只管单字字形，**它过了不等于词对**——同形字（社会・経済・分析）字形对、但可能词不对，进第②关。

## 第③关 表达·语序·标点（直译腔 → 地道）

### 范畴词冗余（中文爱加范畴词，日文删）

- 環境保護**工作** → 環境保護｜経済発展**方面**において → 経済発展において｜失業**問題**の現象 → 失業（という現象）｜「…という**問題**」滥用 → 多数可删

### 中式搭配 → 日式搭配

| 中文直译 | 地道日语 |
|---|---|
| 起到了重要作用 | 重要な役割を**果たした** |
| 具有重要意义 | 重要な意義を**もつ** |
| 随着…的不断发展 | …の発展に**伴い**（删「不断」冗余） |
| 对于…来说 | …に**とって** |
| 通过…来… | …**によって**／…**を通じて** |
| 基于… | …**に基づいて**／…**を用いて** |
| 针对… | …**に対して**／…**を対象に** |
| 从…角度 | …の**観点から**／…の**視点から** |
| 综上所述 | **以上より**／**以上のことから** |
| 越来越… | **ますます**／**次第に** |
| …等等 | …**など** |

### 语序·句构

- 中文流水长句（逗号串到底）→ 日文**断句 + 接続詞**（「一方，」「しかし，」「したがって，」），或合为一个带「〜が，〜であり，〜と考えられる」的多重修饰长句（见文体层）。
- 中文「把/被」字句 → 日文**受身/能动**自然形（「Xを分析した」「Xが示された」）。
- 「进行+分析/研究」可保留为「分析を**行う**」（地道），但「进行了详细的分析」宜改「**詳細に**分析した」。

### 标点·标记符号（净化层必查，逐项替换）

| 中文 | 日本 |
|---|---|
| 逗号 `，` / 顿号 `、` | 按目标刊：理科/计量系 **，．**；人文/政策系 **、。**（见文体层，**不可混用**） |
| 句号 `。` | `．`（理系刊）或 `。`（人文刊） |
| 书名号 `《经济研究》` | 期刊/书 **『経済研究』**；论文/章/篇 **「…」** |
| 引号 `"…"` | **「…」**；内层引号 **『…』** |
| 破折号副题 `——…——` | **──…──** 或 **―…―** |
| 省略号 `……` | **……**（日文亦六点，但居中） |

## 在「日语优化步骤」里的自查流程

1. 跑 `zh_glyph_check.py`（第①关）→ 清字形残留，复核引用/专名区。
2. 拿第②关对照表**逐段扫词**，重点盯 ⚠️ 同形异义陷阱与 E 类官腔。
3. 拿第③关**逐句过表达/语序**，统一标点二分与标记符号。
4. 三关清完，再进 [ja-学術文体.md](ja-学術文体.md) 按投稿/修論轨逐节优化文体。

> 护栏：本关只改「中文味」，**不动论证、数据、引用内容**；引用题名/专名一律按原文保留（即便是中文字形）。拿不准是不是该保留 → 标记问用户，别擅改。
