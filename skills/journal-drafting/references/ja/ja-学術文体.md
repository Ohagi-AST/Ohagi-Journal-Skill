# 日本語学術文体（投稿轨 / 修論轨）

> 母版规则 7 文体层的展开，**仅在日语轨启用**。从经济学日文语料蒸馏（投稿轨 65 篇 J-STAGE+『経済研究』；修論轨 5 篇上智/明治博论）。

> 写作信息密度/主张克制等通用原则仍归 [../en-economics-writing-style.md](../en-economics-writing-style.md)；本文只管**日语学术特有的文体规范**。

## 一、两轨通用（先立这些）

### 1. 文体 = である体（常体）
全篇だ・である体，**绝不混です・ます**。高频句尾：
`〜である／〜となる／〜と考えられる／〜を明らかにする／〜と論じる／〜が示された／〜であろう／〜と推察される`

### 2. 自称（⚠️ 中文习惯最易翻车）
- 研究本体：**本稿・本研究・本論文**（投稿轨多用「本稿/本研究」，修論轨多用「本論文/本章」）。
- **忌「私」入正文**。书评用「**評者**」；谢辞/谦辞用「**筆者**」（「本論文の誤りはすべて筆者に帰属する」）。
- 修論「謝辞」是例外区，可自然用「私」（见三·5）。

### 3. ヘッジ（主张谨慎，日文普遍弱于中文直断）
`〜と考えられる／〜と思われる／〜であろう／〜のように思われる／〜を示唆する／〜と整合的である` 高频。
中文「证明了/表明X导致Y」的强断 → 日文降一档：「**〜が示唆される／〜と整合的である**」。强度纪律仍守母版规则 4（主张强度 ≤ 识别强度）。

### 4. 引用：著者（年）
`Kuroda (2010) によると…／植田（1992）は…としている／松尾（2019）は…と論じている`。
多篇并列：`(金本，1997；速水，2006など)`（顿号/分号 + など）。西文名罗马字：`Clark (2005)，Fama-French`。

### 5. 统计表述（固定说法）
`統計的に有意／1％・5％・10％有意水準／頑健標準誤差／クラスター標準誤差／***，**，*／係数は有意に正（負）`。

### 6. 接続詞（句首高频，按目标刊配标点）
`一方，／他方，／しかし，／つまり，／さらに，／したがって，／具体的には，／これにより，／その結果，`。
靠**实质**推进、别滥用（同母版风格层）。

### 7. 句子结构
- 长句多重修饰：`〜が，〜であり，〜と考えられる`。
- **体言止め・受身**常用：结果陈述爱用受身——`〜が示された／〜が明らかにされた／〜が確認された／〜が観測された`。

### 8. ⚠️ 标点二分（按目标刊惯例，**不可混用**）
- **理系/计量系**：全角 **，．**（jbef・jjce・peq・economia・reeps・prifr 等）。
- **人文/政策系**：**、。**（jjhep・keidaironshu・seikatsukeizaigaku 等）。
- 修論：随**学校规程/学科惯例**，全篇统一。先确认目标刊/规程属哪派，再落笔。

### 9. 标题副题
破折号副题：`──…──` 或 `―…―`，或冒号。例：`地域犯罪に関する実証分析──都道府県別データを用いた外国人犯罪費用──`。

## 二、投稿轨（期刊投稿）

### 要旨/要約
开头 **「本研究では／本稿は／本論文では」** + 背景 → 方法/データ → 結果 → 含意。
例（jbef 体）：「本研究では…独自調査を行い…明らかにする。また…政策評価も行う。分析の結果…」。
末尾附 **「キーワード：」「JEL Classification：」**。

### はじめに（结构定式）
社会/制度動向 → 先行研究 → **問い**（「それではなぜ…だろうか」）→ 本稿の目的 → **構成予告**（「本稿の構成は以下の通りである。まず2節で…，3節で…」）。

### 节构成（典型）
`はじめに → 先行研究 → データ／分析モデル → 分析結果 → 考察 → おわりに`。一篇一主线，篇幅紧。

### 谢辞
放脚注/末尾：「…コメントをいただいた」「記して感謝申し上げたい」「本論文の誤りはすべて筆者に帰属する」。

### 体例
字数/格式/匿名化/投稿样式以**目标刊征稿要求**为准（属母版投稿体例环节，本文不锁死）。

## 三、修論轨（修士/博士论文正文）

### 1. 章立て（论文集合型 — 日本博论主流）
`序章/第1章（総論）→ 第2〜N章（各実証章）→ 終章/結論（総括）`。
**各実証章 = 一篇自足的论文**，各自带 `はじめに → 分析モデル → 分析データ → 分析結果 → 考察 → おわりに(まとめに)`（甚至各自的 付録）。
> 语料实证：森田論文 第1章理論+目的、第2–4章各実証各自まとめに；功刀論文 第1章総論+各章概要、第2–5章各実証。

### 2. 第1章/序章（双职能）
①**理論/概念考察**（広い問題提起 → 基礎文献 IPCC・Samuelson 等 → 概念整理）；
②**本論文の目的**（「本研究ではそうした…に注目し，経済学的視点から分析を行う」）；
③**各章の概要**（roadmap：逐章一段交代各章做什么——修論必备，比投稿轨的「構成予告」更长更详）。

### 3. 各章内部 & 先行研究
- 每个实证章**自带 先行研究 节**（不止一个全局综述）；第1章担**理論的考察**，实证章担**聚焦型先行研究**。
- 結果陈述同样受身爱用：`〜が示された／〜が明らかにされた`。

### 4. 終章/結論
**総括**——跨章汇总各章发现；**今後の課題**（「…は今後の議論としたい」）。结论章承担全局收束，比投稿轨「おわりに」厚。

### 5. 謝辞（修論专属区，文体可松）
- 长、逐一列名：**所属 + 姓 + 敬称（教授/准教授/先生）**，「多くのご支援とご指導を賜りました」「深く感謝申し上げます」「ご教示頂き」「温かく見守って下さり」。
- **此区可自然用「私」**（「未熟であった私を」），与正文忌私相反。
- 收尾谢家人（「両親」「姉兄」「家族」），署**日付 + 氏名**。

### 6. 冗長許容·详尽
修論容忍远多于投稿轨的铺陈：详细 **記述統計（表）**、逐步推导、**付録/Appendix**、数式多者列 **記号一覧表**（「本論文で使用する記号の意味」）。投稿轨要砍的细节，修論可留。

### 7. 自称
正文以 **本論文・本章・本研究** 为主（「本論文では…」「本章では…」）。

## 四、两轨差异速查

| 维度 | 投稿轨 | 修論轨 |
|---|---|---|
| 整体结构 | 单篇一主线 | 序章 + 各実証章（各自足）+ 終章 |
| 第1节职能 | はじめに（紧凑构成预告） | 序章/第1章（理論考察 + 目的 + **各章概要**） |
| 先行研究 | 通常一处 | 第1章理論 + **各章各自聚焦** |
| 收束 | おわりに（短） | 終章/結論（**総括 + 今後の課題**，厚） |
| 篇幅 | 紧，砍冗余 | **冗長許容**（記述統計/付録/記号表） |
| 主自称 | 本稿/本研究 | 本論文/本章 |
| 「私」 | 全文忌 | **謝辞区可用** |
| 谢辞 | 脚注短句 | 長·逐名·署日付氏名 |
| 标点/体例依据 | 目标刊征稿要求 | 学校规程/学科惯例 |
| 主张强度 | ヘッジ + 投稿审稿压低 | ヘッジ + 答辩委员视角 |
