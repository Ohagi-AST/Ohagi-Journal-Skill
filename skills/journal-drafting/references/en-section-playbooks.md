# 经济学论文分节 Playbook（按需加载）

> 用途：给标题、摘要、引言、模型、数据、结果、结论与附录提供默认结构。目标刊真实论文、官方要求或 `journal-adapt` 动态层与本文件冲突时，前者优先。
> 护栏：缺少作者事实、估计值、样本信息或引用时，写明确占位符 `【作者补：…】` / `【待核】`，绝不补造。

## 目录

1. 先按论文类型选模板
2. 标题

## 1. 先按论文类型选模板

- **应用实证**：问题 → 数据与制度 → estimand/识别 → 主结果 → 威胁导向稳健性 → 异质性/机制 → 边界。
- **纯理论**：现实谜题 → 最简环境 → 均衡/命题 → 经济直觉 → 比较静态 → 扩展/可检验含义。
- **理论+实证**：每个实证检验必须回指某个模型预测；不能让模型与回归各写各的。
- **结构估计**：模型 → 识别与估计 → 拟合/外部验证 → 反事实与福利 → 假设敏感性。
- **描述/测量**：先明示“不识别因果效应”；数据构造、测量有效性与新事实本身就是贡献。

## 2. 标题

- 让非本领域经济学者一眼看出经济对象、核心关系或机制；上下文确有辨识价值时再加场景副题。
- 不以常规方法名领头，除非方法本身是贡献；不用 `A Study of…`、`Various Factors…` 一类空泛标题。
- 评价维度：清楚、具体、与真实贡献一致、容易区分；不把固定词数当硬门槛。
- 信息不足时给 2–4 个方向性候选，并标明每个候选仍缺什么事实；不要替论文发明结论。

## 3. 摘要

默认四块，顺序可按目标刊调整：

1. **做什么**：研究问题或理论洞见。
2. **怎么做**：数据+识别，或模型+机制。
3. **发现什么**：主结果、量级与不确定性；缺数字就标作者补，不写“有显著影响”。
4. **为何重要**：只写证据支持的含义。

写作时先完成引言和主结果叙事，再写摘要。字数、关键词、JEL 与披露要求只服从目标刊当前官方规则。摘要通常不做文献综述；若目标刊或谜题需要一个前作锚点，再保留最少必要信息。

## 4. 引言

- 首屏让读者知道问题、做法和核心发现；可从大事实、谜题、争议或规模切入，禁止清嗓子式文献史。
- 默认顺序：`Hook/Stakes → Question → Data+Design/Model → Main findings → Value added relative to closest work → Roadmap（确有帮助才留）`。
- 结果用真实量级与不确定性，不以“interesting/significant results”代替。
- 文献定位讲关系，不逐篇报摘要；是否设独立文献节、篇幅多少，由论文类型和目标刊母本决定。
- contribution 对最近邻逐条落差：数据、识别、测量、机制、模型或反事实，不能只写“fills a gap”。

## 5. 模型与理论

1. 数学前先用自然语言交代经济环境、参与者、时序、信息与核心摩擦。
2. 列 primitives、约束与均衡概念；每个假设标注“关键识别/机制”或“仅为简化”。
3. 命题先给大白话含义和机制，再给形式表述；证明只有在帮助理解机制时留正文。
4. 解释比较静态由哪条激励、约束或权衡驱动。
5. 逐一放松关键假设；理论+实证论文把预测映射到具体回归或 exhibit。
6. 只编号后文会引用的公式、命题和引理；变量首次出现即定义，符号全篇一致。

## 6. 数据与制度背景

- 首段交代数据源、版本/时间、地理范围、观测单位与获取方式。
- 给出样本构造：纳入/排除、合并键、匹配率、缺失处理、异常值/缩尾、最终样本量。
- 精确定义处理、结果、控制、固定效应与单位/变换；重要限制放正文，不埋脚注。
- 多数据源说明合并顺序和未匹配样本；受限数据说明申请路径、限制与可复现边界。
- 描述统计表须自含；RCT/准实验的平衡性与样本流失按设计单列。

## 7. 结果、异质性与机制

- 主结果先行，并点名其 estimand、识别变异和对应 exhibit；从最透明设定走向更丰富设定。
- 分开报告方向、量级、不确定性、经济意义与边界；只用数据中真实存在的基准换算。
- 稳健性按“具体威胁 → 对应检验 → 结论改变多少”组织，不堆规格。
- 异质性先给经济理由和预设状态，再报告组间正式检验与多重检验处理；不能用“一组显著、一组不显著”推出组间差异。
- 机制用三步：理论预言渠道 M → 若 M 成立应观察 X → 数据是否支持 X。未直接识别时称“提示性渠道证据”。

## 8. 零结果

- 区分**精确地接近零**与**估计不精确**。`p > 0.05` 本身不能证明没有效应。
- 报告置信区间，并说明能排除多大的经济相关效应；必要时说明设计功效或最小可检测效应。
- 预注册结果照实报告；不得改写成“边际显著”或隐藏不利结果。
- 把零结果放回理论和既有证据：它排除了什么、仍允许什么，不把失败包装成贡献。

## 9. 结论

- 用不同于摘要/引言的表述总结最核心发现，不逐表复述。
- 推论必须在识别、外部有效性和模型假设范围内；政策含义写受益者、成本与适用条件，缺成本数据就不做成本收益结论。
- 重要限制要诚实且具体，不做削弱全文的通用 caveat 清单，也不压下真实限制。
- 未来研究只写由当前结果自然产生、可执行的 1–2 个方向；不以“more research is needed”收尾。

## 10. 附录

- 主文独立成立；核心 estimand、识别逻辑与头牌 exhibit 不得流放附录。
- 附录按主文顺序组织，容纳证明、数据构造、额外稳健性、安慰剂和完整结果。
- 每个附录项目都从主文准确指向；编号、文件名与生成脚本保持可追踪。
