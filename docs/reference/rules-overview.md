# 母版规则概要（Rules Overview）

母版 [`template-master-framework.md`](../../skills/journal-drafting/template-master-framework.md) 的「通用硬规则 1–9」一页速查。**权威表述以母版为准**，本页只为快速定位。

> 设计来源：把 agent harness 的 *Rules-as-spec（1–7）+ Verification-loop（8）+ Memory（9）* 落到论文成稿。

## 第 0 层｜每篇填空的 SPEC（Phase 0，不填满不动笔）

带 ⛔ 的项空着 = 禁止进入正文写作：

- ⛔ 目标期刊（具体刊名）
- ⛔ 目标层级现实性（贡献/数据/识别诚实对得上吗，对不上就降档）
- ⛔ 结构母本 3–5 篇
- 一句话研究问题、一句话 contribution、本篇专属红线、写作语言、引用样式、数据出处。

## 通用硬规则 1–9

| # | 规则 | 一句话 |
|---|---|---|
| 1 | 叙事框架 | 一篇 = 一个核心问题；Intro 是填配方不是创作；每节开头一句"做什么、得什么"。 |
| 2 | 文献综述 = 定位 | 每簇收束到一句 gap；产出「文献地图」表，先有表再写散文；禁逐篇罗列。 |
| 3 | 新规性 / 贡献 | 1–3 条显式 bullet；每条对应真实检索；⛔ 禁裸 "we are the first to"；扛得住点名最近邻。 |
| 4 | **主张强度纪律** ★ | **结论语气 ≤ 识别强度**。关联/描述性设计全文禁因果措辞；配「识别审计清单」+ 设计专用补检。 |
| 5 | 引用诚信 | ⛔ 绝不编造作者/年份/标题/DOI；拿不准标【待核】；双时点核验（地基引用早查 + 投稿前穷尽）。 |
| 6 | 数据与可复现 / 图表先行 | 先设计整套图表再写散文；全文一套 house-style（三线表、禁九宫格）；视觉自检；**渲染≠造数**。 |
| 7 | 文体 / 语域 | 简洁、主动语态、发现用现在时；hedging 纪律；⛔ 内部暗语禁入正文。 |
| 8 | 验证闸 | 《单节自查》每节跑；《总闸》投稿前跑；再跑引用穷尽核验 + 审稿人预演。 |
| 9 | 防漂移 / 记忆 | 每篇一份「成稿决策日志」；顶部钉死一句话问题；重大转向留痕。 |

## 三道验证闸

1. **《单节自查》**（每写完一节）——强度≤识别、引用真实、无内部暗语、图表过视觉自检。
2. **《总闸》**（投稿前）——研究问题没漂、贡献扛得住、无残留【待核】、无 over-claim、格式合规。
3. **《审稿人预演》**——以目标刊 referee 视角产出 Summary / Strengths / Major / Minor / Recommendation / 修改计划。

## 适用档位

期刊投稿档全配置跑；学位论文档自动跳过/软化期刊专属环节（动态写作层、层级现实性、新规性穷尽、投稿体例/字数硬限/复现包），**通用骨架（规则 1·2·4·5·7·8·9）照走**。
