# 领域扩展模板（Field Templates）

不同子领域的呈现惯例不一样（宏观 / 金融 / 贸易引力 / 理论 / 结构…）。母版规则 1 与 [`en-field-conventions.md`](../../../skills/journal-drafting/references/en-field-conventions.md) 已收了几个；要把**你自己的领域写作惯例**加进来，照下面这张模板填一节，追加到 `en-field-conventions.md`。

## 为什么要它

通用骨架（规则 1）是所有论文共用的；但「这一类论文谁领头、表怎么排、用什么识别、引哪些经典」是**领域特有**的。把它沉淀成可复用的一节，下次同领域起草直接套，不必每篇重想。

## 模板结构

复制下面这块，把方括号填实：

```markdown
### [领域名称]（如：环境经济学 / 公司金融 / 国际贸易引力）

- **常用方法 / 识别策略**：[这个领域最常见的识别设计——DID / 事件研究 / IV / RDD / 结构估计 / 校准…；
  以及该领域特有的识别痛点，如引力模型的"FE 之后剩什么变异在识别"。]
- **图表呈现惯例**：
  - 头牌（领头对象）：[基准系数表走列 / 主图（系数图·事件研究图）/ 核心命题 / 校准表]
  - 表/图偏好：[这个领域更偏好表还是图？主结果通常是什么形态？]
  - house-style 细节：[星号约定、SE 类型、是否报 baseline mean、特有的稳健性呈现…]
- **写作风格要点**：[这个领域审稿人特别在意什么？常见的 over-claim 雷区？必备的 caveat？]
- **必引经典 / 引用样式**：[绕不开的奠基性文献类型；引用样式如 AEA author-date。]
```

## 一个填好的例子（节选自现有约定）

```markdown
### 贸易引力（Gravity）

- **常用方法 / 识别策略**：PPML 估计 + 高维固定效应（出口国×年、进口国×年、国家对）。
  ⛔ 识别痛点：固定效应之后必须说清"还剩什么变异在识别系数"，否则 referee 一票否决高发。
- **图表呈现惯例**：
  - 头牌：基准系数表（走列：逐列加 FE / 换样本）。
  - 表/图偏好：主结果以**表**领头；弹性/异质性可配系数图。
  - house-style：报 N、伪 R²；星号三级 0.10/0.05/0.01。
- **写作风格要点**：避免把"贸易成本下降"直接说成因果，除非政策冲击提供了外生变异。
- **必引经典 / 引用样式**：引力理论奠基作 + PPML 方法作；AEA author-date。
```

## 加进来之后

1. 把你的新一节追加到 [`en-field-conventions.md`](../../../skills/journal-drafting/references/en-field-conventions.md)。
2. 若该领域有特殊的图表头牌方式，可在 [`en-exhibit-design-narrative.md`](../../../skills/journal-drafting/references/en-exhibit-design-narrative.md) 的「头牌按方法族」处补一行。
3. 逐节写到结果/识别章时，agent 会按需读这份约定，不会硬套通用骨架。
